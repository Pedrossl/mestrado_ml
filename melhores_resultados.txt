LEGENDA DE MÉTRICAS
  Acc   = Acurácia — % total de predições corretas (enganosa em dados desbalanceados)
  Sens  = Sensibilidade (Recall) — % de casos positivos detectados corretamente
  Spec  = Especificidade — % de casos negativos classificados corretamente
  F1    = F1-Score — média harmônica entre Precisão e Sensibilidade
  Kappa = Concordância de Cohen (0.21–0.40 = razoável; >0.40 = moderada)
  IC95  = Intervalo de Confiança 95% (±)
  (*)   = Acc calculada via Acc = Sens×0.153 + Spec×0.847 (proporção do dataset)

  GAD: 1. XGBoost (Kappa=0.372) >> 2. SVM (0.210) >> 3. ADTree (0.156)
  SAD: 1. XGBoost (Kappa=0.256) ≈ 2. SVM (0.255) >> 3. ADTree (não avaliado)

LIMITAÇÕES PRINCIPAIS:
  - Dataset pequeno (n=287; 44 GAD+, 46 SAD+) → ICs amplos, especialmente em Sens
  - Nenhum modelo atingiu Kappa moderado (>0.40) para nenhum target
  - Alta variabilidade nos folds (Sens varia ±10–19pp dependendo do modelo)
  - Acc é métrica enganosa aqui: modelo que chuta tudo negativo tem Acc=84.7%
  - Modelos não substituem avaliação clínica: servem como ferramenta de triagem
